안녕하십니까 교수님
컴퓨터정보과 3-Q 201644048 김응빈입니다.

기본적으로 초기데이터는 존재하지 않고, 사용자가 직접 입력하는 방식입니다.
이수에는 전공/교양만 입력받도록, 필수에는 필수/선택만 입력받도록,
학점, 출석점수, 과제점수, 중간고사, 기말고사 모두 숫자만 입력받도록 하였고
학점은 1~3까지만 입력 가능하도록, 출석점수와 과제점수는 0~20까지만 입력 가능하도록,
중간고사와 기말고사는 0~30까지 입력 가능하도록 예외처리를 해두었습니다.

그리고 교양과목은 이번년도 1학기의 모든 교양과목이 P/NP라는 말이 있어 교양과목은
무조건 P/NP로 처리하였습니다.

NP와 F는 빨간색글씨로 처리하였습니다.

테이블 정렬은 각 컬럼[열]을 클릭하시면 정렬이 가능합니다.
또한, 중복정렬도 가능합니다.
예를들면, pdf파일에서의 과제내용인 이수별로, 필수별로, 과목별로 정렬하라는 내용이 있었습니다.
shift+마우스좌클릭으로 이수, 필수, 과목명을 순서대로 누르면 중복정렬이 가능합니다.
중복정렬 ==> shift+클릭
